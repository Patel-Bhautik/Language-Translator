# Password-Generator
* Language Translator is a versatile tool designed for translating text and documents across multiple languages using the free Google Translate API. This project provides a user-friendly, efficient translation experience, supporting various document formats and batch translation requests for quick, streamlined results. Ideal for personal or small-scale translation needs, Language Translator makes multilingual content accessible with minimal setup.

Features
* Text Translation: Translates text easily between multiple languages with the free Google Translate API.
* Document Translation: Upload entire files (TXT) for automatic translation, retaining the original format and structure.

Supported Document Types
* TXT – Plain Text File

Installation
* Clone the repository:
git clone https://github.com/Patel-Bhautik/language-translator.git

Document Translation
* To translate a document, upload it via the designated endpoint or user interface (if provided). Supported formats include TXT and the translated output will retain the original format.
